# shorts-editor（アフレコ→縦型ショート動画エディタ）

アフレコ音声と素材フォルダから、テロップ入りの縦型ショート動画（リール）を作るツールです。
ブラウザ上のエディタ（http://localhost:8766）でカードごとに素材・テロップ・演出を編集できます。

## セットアップ（初回のみ）

Homebrew が必要です（未導入なら https://brew.sh から）。

```bash
./install.sh
```

ffmpeg・Python環境・依存パッケージを自動でセットアップします。

## 使い方

```bash
./start.sh <プロジェクト名>
```

ブラウザで http://localhost:8766 が開きます。カードごとに素材の割り当て・テロップ・
フォント・装飾・頭の演出などを編集し、「動画出力」で書き出せます。

## HDR撮影素材の色について（任意）

iPhoneの「HDRビデオ」で撮った素材（HLG）は、そのままだと色がずれることがあります。
このツールは書き出し時に自動で色を補正しますが、**より正確に補正するには zscale対応の
ffmpeg が必要**です。正確にしたい場合だけ、次を実行してください（ffmpegの再ビルドに
時間がかかります）:

```bash
./install_hdr_ffmpeg.sh
```

- 実行しなくてもツールは動きます（HDR素材は近似補正になります）。
- **一番確実なのは、iPhoneの「設定 → カメラ → ビデオ撮影 → HDRビデオ」をオフにして
  撮影すること**です（最初からSDRになり、色補正が不要になります）。
